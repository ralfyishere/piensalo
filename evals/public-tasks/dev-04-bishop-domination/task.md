# Task: Minimum dominating set of bishops

On a standard 8x8 chessboard, a bishop attacks along both diagonals. A set of
bishops **dominates** the board if every square is either occupied by a bishop
or attacked by at least one bishop.

Determine the **minimum number of bishops** required so that every square of
the 8x8 board is occupied or attacked (the bishop domination number of the
8x8 board). Give a brief justification: why that many suffice, and why fewer
cannot.

Deliverable contract — one line exactly:

    MIN_BISHOPS: <integer>

Put your justification in prose above that line. The line is parsed by a
script; get the format exact.
