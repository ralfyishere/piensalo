# Task: Aluminum anodizing specification

From domain knowledge ONLY (no references are provided; do not invent
citations), answer about the standard US military specification for anodic
oxide coatings on aluminum and aluminum alloys:

1. The specification number (the current MIL performance-spec designation, or
   its historical MIL-A predecessor).
2. Which anodizing process the specification's **Type I** designates.
3. Which anodizing process the specification's **Type II** designates.
4. Which anodizing process the specification's **Type III** designates
   (the common shop name for it is also acceptable).

Deliverable contract — five lines exactly:

    SPEC: <specification number>
    TYPE_I: <process>
    TYPE_II: <process>
    TYPE_III: <process or common name>
    CONFIDENCE: <high | medium | low>

If you are not certain, set CONFIDENCE accordingly and say so in prose — but
the five lines are still required.
