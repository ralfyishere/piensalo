# Task: Write a release announcement under a strict contract

Write a short user-facing release announcement for version **2.4** of the
(fictional) note-taking app **Lumen Notes**. The release facts you must
convey, accurately:

1. Notes now **sync offline** and merge automatically on reconnect (new).
2. Search is **40% faster** on large notebooks.
3. Support for **Android 8** ends on **September 30** — users should update
   before then.

Deliverable contract for your answer file:

- First line exactly `HEADLINE: <headline text>` (max 80 characters of
  headline text, column 0).
- A line exactly `AUDIENCE: existing-users`.
- Body prose conveying all three facts (free form, no Markdown bold, no
  code fences).
- Last line exactly `CTA: Update Lumen Notes in your app store`.

The HEADLINE, AUDIENCE and CTA lines are parsed by a script; get them exact.
