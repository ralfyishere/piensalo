# Near-miss: the skill should NOT fire

**Task:** "What year was the HTTP/2 spec published?"

This is a single-fact lookup with one authoritative source and no conflicting evidence. There is nothing to reconcile, grade, or map.

**Why firing would hurt:** running the full pipeline (claim decomposition, source grading, contradiction map, evidence table) on a one-fact question adds latency and produces ceremony — a table wrapped around a single date — while diluting attention that a direct answer with a source would have served better.
