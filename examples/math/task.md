# Task

A packaging machine is bought for **$40,000**. It depreciates **20% per
year** (on its current book value) for **3 years**. A refurbishment then
raises its resale value by **15% of the depreciated value**. What is the
resale value after refurbishment? Show working; end with a line
`RESALE_VALUE: <amount, two decimals>`.
