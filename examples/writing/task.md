# Task

Write the customer email about yesterday's outage. Constraints from support
lead: **maximum 150 words**; must state the outage window (14:07-14:52 UTC,
checkout only), that card data was never at risk, the concrete remedy
(orders stuck in "pending" were auto-cancelled and refunded within 24h), and
a named contact (Maya in support). No technical jargon — customers are
non-technical. Subject line included.
