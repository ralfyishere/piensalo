# Positive example

**Proposal (growth planning):** "Send re-engagement emails to cut churn."

**Trigger present:** an intervention is proposed with no stated path from action (email) to outcome (lower churn).

**Procedure applied:**
- Chain written: email sent → user opens (ASSUMED: ~20% open rate, from memory) → user re-engages (ASSUMED) → churn drops (ASSUMED). Zero verified arrows.
- Confounder named: cancellations may follow billing failures — a side path (billing → churn) the email cannot touch.
- Weakest arrow: re-engagement → churn drop. If churners were never disengaged, the whole chain collapses.
- Check run first: query whether churned users were active in the 7 days before cancelling. Result: 81% were active.

**Required output produced:**
The labeled chain, the billing side path, and the weakest-arrow check — which showed churn is not disengagement-driven. The email intervention targets a non-cause; it is not endorsed, and the billing path becomes the next hypothesis.
