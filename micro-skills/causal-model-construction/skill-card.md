# causal-model-construction — skill card

## What it does
Before an intervention is committed, writes the explicit causal chain from action to intended outcome as named nodes and arrows, labels each arrow verified or assumed, names confounders and side paths for the assumed arrows, and checks the weakest load-bearing arrow first.

## Trigger
- The task involves how a system produces its behavior: "why is/does", root cause, "what's causing/driving", mechanism questions.
- An action is proposed with no written path from action to outcome.
- Post-draft signal: the draft recommends an intervention without an articulated causal chain.

## Counterindications
- The action is instantly reversible and near-free — direct trial dominates modeling.
- A causal chain with labeled arrows already exists from earlier in the session.
- Pure observation tasks with no intervention.

## Negative-transfer risk
Medium distraction risk. Main failure mode: modeling overhead on cheap reversible actions where trying is faster than theorizing — the counterindications route those to direct trial. Do not let chain-writing delay interventions whose cost of being wrong is negligible.
