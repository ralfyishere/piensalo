# Benchmark status — causal-model-construction

Fable Think grades every skill on one evidence ladder:

- **DESIGNED** — specified from documented reasoning-failure modes; never executed as a packaged skill.
- **SMOKE_TESTED** — executed end-to-end in live sessions; behaves as specified; no measured lift claimed.
- **EXPERIMENTALLY_TESTED** — showed a measured effect in at least one controlled comparison.
- **REPLICATED** — the measured effect reproduced in an independent run.
- **PROMOTED** — replicated effect plus demonstrated generalization beyond the original task family.
- **NARROW** — real measured effect, but confined to a specific task family; use only there.
- **REJECTED** — tested and showed no effect or a negative effect; retained for the record.
