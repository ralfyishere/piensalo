# Near-miss example — no external spec exists

**Task:** "Jot down some rough thoughts on how we might restructure the onboarding email — just exploring, no format needed."

There is no spec, rubric, or submission rule set anywhere in the task — the "no external spec" counterindication. The skill must NOT fire.

**Why firing would hurt:** With nothing to disqualify against, the scan would invent phantom rules ("emails must have a subject line under 60 characters?") and check the draft against requirements the user never stated — pure overhead that also risks distorting an intentionally loose exploration.
