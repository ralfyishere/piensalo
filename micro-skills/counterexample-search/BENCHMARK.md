# Benchmark status — counterexample-search

Fable Think grades every skill on a fixed evidence ladder:

- **DESIGNED** — specified from documented failure modes; not yet executed as a packaged skill.
- **SMOKE_TESTED** — ran end to end at least once; no controlled comparison.
- **EXPERIMENTALLY_TESTED** — showed an effect against a baseline in at least one controlled run.
- **REPLICATED** — the effect held in an independent re-run.
- **PROMOTED** — replicated with an effect size that justifies default use.
- **NARROW** — real effect, but confined to a specific task class; use only there.
- **REJECTED** — tested and showed no effect (or harm); kept for the record.
